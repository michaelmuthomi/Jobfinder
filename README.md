## Usage
  > Prerequisite
  > 
  > xampp
1. Fork the repo
2. Clone the repo
```
git clone https://github.com/<your username>/Jobfinder
```
3. cd into the repo
  ```
    cd Jobfinder
  ```
4. Create a Database called
  ```
    Jobfinder
  ```
5. Import The SQL file in
  ```  
    jobfinder/database/jobfinder.sql
  ```
6. In your browser run
  ```
  localhost/jobfinder
  ```
