## Usage
1. Download / Clone The repository
2. cd into the repo
  ```
    cd Jobfinder
  ```
3. Create a Database called
  ```
    Jobfinder
  ```
4. Import The SQL file in
  ```  
    jobfinder/database/jobfinder.sql
  ```
5. Run the Project in this url
  ```
  localhost/jobfinder
  ```
